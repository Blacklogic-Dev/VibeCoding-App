Module 1: Introduction to Machine Learning

Machine learning teaches machines to learn patterns from data.

from sklearn.datasets import load_iris
from sklearn.tree import DecisionTreeClassifieriris = load_iris() model = DecisionTreeClassifier() model.fit(iris.data, iris.target)

Module 2: Supervised Learning

In supervised learning, models learn from labeled data.

from sklearn.linear_model import LinearRegression X = [[0], [1], [2]] y = [0, 1, 2] model = LinearRegression().fit(X, y)

Module 3: Unsupervised Learning

Clustering and grouping unlabeled data.

from sklearn.cluster import KMeans X = [[1, 2], [1, 4], [1, 0], [10, 2], [10, 4]] model = KMeans(n_clusters=2).fit(X)

Module 4: Model Evaluation

Evaluate model accuracy and performance.

from sklearn.metrics import accuracy_score y_true = [0, 1, 1, 0] y_pred = [0, 1, 0, 0] print(accuracy_score(y_true, y_pred))
