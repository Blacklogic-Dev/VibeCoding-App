Module 1: Introduction to HTML

HTML is the structure of web pages. It uses tags to define elements like headings, paragraphs, and links.

<!DOCTYPE html>
<html>
  <head><title>My Web Page</title></head>
  <body>
    <h1>Welcome to VibeCoding</h1>
    <p>Let the code speak.</p>
  </body>
</html>

Module 2: Styling with CSS

CSS styles your page with colors, layout, and fonts. It makes your HTML beautiful.


body { background-color: #000; color: #66ccff; font-family: Arial; } h1 { color: gold; }

Module 3: Introduction to JavaScript

JavaScript adds interactivity to your web pages. Use it to respond to user actions.


document.querySelector("h1").onclick = function() { alert("Welcome to VibeCoding!"); };

Module 4: Responsive Layouts

Use CSS Flexbox/Grid and media queries to make your site look good on all screens.


@media (max-width: 600px) { body { background-color: #222; } }

Module 5: VibeCoding Web Prompt

"Build a 3-section landing page with HTML for layout, CSS for design, and JavaScript for interactivity."
