Module 1: Java Basics

Module 2: JavaScript Basics

JavaScript is dynamic and runs in browsers. Used for interactivity.

Module 3: Java Conditions

Use if/else and switch to make decisions.

int age = 20;
if (age >= 18) {
  System.out.println("Adult");
} else {
  System.out.println("Minor");
}

Module 4: JavaScript Events

Events react to user actions.

  alert("Clicked!");
};
